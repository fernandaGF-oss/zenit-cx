    Configurar corredores

    Un corredor agrupa varios servicios entre dos zonas (ej: Paraná - Córdoba puede incluir múltiples rutas). Agregá los corredores que querés usar como filtro.

      Agregar agencias manualmente

      Las agencias del filtro salen automáticamente del OCR de los pasajes. Si querés agregar agencias que todavía no aparecen en respuestas, o variantes del nombre que el OCR lee distinto, sumalas acá.

      Coincidencia parcial: si agregás "LINIERS", el filtro mostrará todas las respuestas cuyo OCR contenga ese texto (ej: "LINIERS BOL.14", "LINIERS BOL 14", etc.).

        Abreviaciones de ciudades

        Cuando el OCR lee "Parana" en un pasaje, el dashboard tiene que convertirlo al código que usa el Excel de servicios ("PNA"). Si una ciudad no está bien mapeada, el cruce con servicios va a fallar para esa ruta.

        Ya hay 46 ciudades cargadas por defecto (Paraná, Retiro, La Plata, Mar del Plata, etc.). Acá podés agregar las que falten o sobrescribir las que estén mal.

        El código debe ser de 3 letras, en mayúsculas. Mirá el Excel de servicios para conocer los códigos reales.
